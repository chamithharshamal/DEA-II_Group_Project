# Healthcare Management System - Group 35

### Project Structure
All backend services must be placed inside the `/backend` folder.

### How to add your service:
1. Go to [start.spring.io](https://start.spring.io).
2. **Project:** Maven, **Language:** Java 17.
3. **Group ID:** `com.nsbm.group35.healthcare`.
4. **Artifact:** (e.g., `patient-service`).
5. Download and unzip the project **directly into** your folder in `/backend`.

### Port Assignments (Required)
To avoid conflicts, update your `src/main/resources/application.properties` with your assigned port:
- admin-service: 8081
- appointment-service: 8082
- billing-service: 8083
- doctor-service: 8084
- lab-report-service: 8085
- notification-service: 8086
- patient-service: 8087
- pharmacy-service: 8088
- staff-service: 8089
